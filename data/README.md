## Datasets

This repository is for the cleaned dataset, as well as the raw files.

-   `students`: contains raw data, see citation section for source

-   `students_clean`: cleaned data, adjusted names, and variables changed to factors

-   `students_codebook`: contains key to data, considering some values are represented as numbers that correspond to meaning.
